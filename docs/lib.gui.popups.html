lib.gui.popups.confirmations module¶

lib.gui.popups.confirmations.exit_wo_save()¶

GUI Popup to Confirm User Exit w/o Save

This function will create a popup window asking the user if they are sure they want to exit without saving. After the user answers the function will return with a boolean idicating whether the user said Yes (True) or No (False)

Example:

confirms = lib.gui.popups.confirmations

#…

if confirms.exit_wo_save():

# Exit, without saving

else:

# Most likely return, changing nothing….

Returns

Boolean
